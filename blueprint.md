# Lotto Number Generator

## Overview

This application generates a set of 6 unique random numbers for a lottery, typically between 1 and 45. It provides a simple, user-friendly interface with a partnership inquiry form.

## Design and Features

*   **UI Components:**
    *   A prominent button to trigger the number generation.
    *   A display area to show the generated numbers with a smooth animation.
    *   A fully functional partnership inquiry form using Formspree for handling submissions.
*   **Styling:**
    *   Modern and clean design with dark/light mode support.
    *   Visually appealing layout with balanced spacing.
    *   Interactive elements with clear feedback.
    *   Responsive design for various screen sizes.
    *   CSS animations for a more dynamic user experience.
