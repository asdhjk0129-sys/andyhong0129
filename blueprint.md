
# Lotto Number Generator

## Overview

This application generates a set of 6 unique random numbers for a lottery, typically between 1 and 45. It provides a simple, user-friendly interface for generating and displaying the numbers.

## Design and Features

*   **UI Components:**
    *   A prominent button to trigger the number generation.
    *   A display area to show the generated numbers.
*   **Styling:**
    *   Modern and clean design.
    *   Visually appealing layout with balanced spacing.
    *   Interactive elements with clear feedback.
    *   Responsive design for various screen sizes.
